# Sound-Equalizer--
 This is a Sound Equalizer GUI App Using Python's PyQt5. It gives you the ability to play, pause, and Equalize any one-channel wav audio file and play 3 different instruments. 
  
 ## Toolbox

-   Python
    -   PyQt5
    -   pyqtgraph
    -   numpy
    -   scipy
    -   matplotlib
